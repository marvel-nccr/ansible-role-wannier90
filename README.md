# Ansible Role: marvel-nccr.wannier90

An ansible role that installs [Wannier90](http://www.wannier.org/) on Ubuntu.

## Installation

`ansible-galaxy install marvel-nccr.wannier90`

## Role Variables

See `defaults/main.yml`

## Example Playbook

```yaml
- hosts: servers
  roles:
  - role: marvel-nccr.wannier90
```
